3. Facebook Connect를 통해 로그인
Facebook Connect를 사용하여 서비스 사이트에 로그인할 수도 있습니다. 이 경우, 당사에 별도의 사용자 계정을 등록할 필요가 없으며 기존 Facebook 계정을 사용하여 로그인할 수 있습니다.
이 과정에서 Facebook은 사용자의 특정 개인 데이터를 당사에 전송합니다. 등록 프로세스 중에 Facebook이 사용자에게 데이터가 전송된다는 사실을 알립니다("Facebook 데이터").
사용자는 그러한 Facebook 데이터 전송에 명확하게 동의하거나 거부할 수 있습니다. Facebook은 사용자의 명시적 허가 없이는 개인 Facebook 데이터를 당사에 전송하지 않습니다.
당사는 사용자의 Facebook 사용자 이름 및 Facebook 공개 프로필을 볼 수 있게 됩니다.
본 개인정보 보호정책에 자세히 설명된 목적 또는 당사가 사용자의 개인 정보를 수집하는 시점에서 사용자에게 설명한 용도로만 이 Facebook 데이터를 사용합니다.
이와는 반대로, Facebook은 사용자가 당사 사이트에 로그인할 때 Facebook Connect를 사용했는지 여부와 시기를 확인할 수 있습니다.

서비스 사이트에 로그인할 때 Facebook Connect 사용을 언제든지 중지할 수 있습니다. 원하는 경우, 당사 사이트에서 본인의 사용자 프로필 계정을 만들 수 있습니다.
참고로, 사용자 계정을 만들지 않아도 서비스 사이트를 이용할 수 있습니다.

5. 쿠키, 웹 비콘 및 유사 기술

이러한 쿠키에는 개인 데이터가 포함되어 있지 않습니다.
사용되는 일부 쿠키는 사용자의 세션이 만료될 때 즉, 사용자가 브라우저를 닫을 때 자동으로 삭제됩니다. 이를 세션 쿠키라고 합니다.
기타 다른 쿠키는 사용자의 디바이스에 계속 남아 있으며, 이후에 사용자가 다시 방문할 때 당사 또는 당사의 비즈니스 파트너가 사용자의 브라우저를 인식할 수 있습니다(영구 쿠키).

사용자가 보다 쉽고 편리하게 서비스 사이트에 액세스하도록 하거나 당사가 특정 서비스를 제공할 수 있도록 추적 기술을 사용하여 사용자에 대한 정보를 수집합니다.
